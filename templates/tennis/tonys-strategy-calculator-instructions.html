Strategy Calculator Instructions

What Is This?

This tool helps you find the optimal strategy when you're in a situation with a few choices, and your opponent is doing the same. It uses a concept called 'Nash Equilibrium' to tell you how often you should pick each of your options to get the best results over time. This prevents your opponent from predicting your moves.

Example: The Serve & Return

Let's imagine a common tennis scenario.

  • Player 1 (The Server) can serve either "Out Wide" or "Down the T".
  • Player 2 (The Returner) can anticipate the serve and cover either the "Wide" side or the "T".

Your job is to input the server's estimated win percentage for each of the four possible outcomes.

  • Server Serves Wide - Returner Covers Wide - Server Wins 45%
  • Server Serves Wide - Returner Covers T - Server Wins 70%
  • Server Serves T - Returner Covers Wide - Server Wins 65%
  • Server Serves T - Returner Covers T - Server Wins 30%

After entering these percentages into the calculator, it will tell you the optimal, unpredictable strategy for both the server and the returner.
