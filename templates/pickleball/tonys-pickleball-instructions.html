Tony’s Tennis Pickleball Tracking Instructions

Core Tracking

  • Return of Serve Tracking is based on the return of serve. For every serve, first or second, log whether the returner (or returning team) won or lost the point. The trackers will automatically calculate serving percentages based on this data.
  • Undo Last Point Made a mistake? This button will remove the very last action you recorded.
  • Game Complete Press this after each game to reset the return-of-serve tallies for the next game.
  • Game Scoring Game scoring is not dynamic. This is on purpose. One, it is easy to miss points when tracking matches. Two, players frequently play the wrong score within a game.

Singles-Specific Tracking

  • Unforced Errors & Second Shot Misses Use these buttons to log unforced errors and second shot misses. (For an unforced error on a second shot, add 1 to both tallies.) The tallies for these are automatically reset at the start of each new set.

Doubles-Specific Tracking

  • Server & Returner Positions
    • Use the dropdown menu to select the current server for each game.
    • Use the "Switch" button to correctly position the returning players or to update their positions if they change returning positions between sets.
  • Second Shot Misses The buttons for second shot misses are labeled with the player's position (S for Server, N for Net, D for Deuce Returner, A for Ad Returner). This helps you track who is missing and from where on the court. This tally also resets at the beginning of each new set.
